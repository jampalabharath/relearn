Aggregate Functions

SQL aggregate functions perform calculations on multiple rows of data and return summary results.

1. Basic Aggregate Functions

COUNT â€“ Count rows

SELECT COUNT(*) AS total_customers
FROM customers;

SUM â€“ Total of values

SELECT SUM(sales) AS total_sales
FROM orders;

AVG â€“ Average of values

SELECT AVG(sales) AS avg_sales
FROM orders;

MAX â€“ Maximum value

SELECT MAX(score) AS max_score
FROM customers;

MIN â€“ Minimum value

SELECT MIN(score) AS min_score
FROM customers;

2. Grouped Aggregations â€“ GROUP BY

Aggregate results per group.

SELECT
    customer_id,
    COUNT(*) AS total_orders,
    SUM(sales) AS total_sales,
    AVG(sales) AS avg_sales,
    MAX(sales) AS highest_sales,
    MIN(sales) AS lowest_sales
FROM orders
GROUP BY customer_id;
